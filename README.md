# Collection of git repo that useful for me and maybe useful for others

### Batch tools
1. [Converseen](https://github.com/Faster3ck/Converseen)
   Batch convert, resize, rotate, and flip images
2. [Caesium](https://github.com/Lymphatus/caesium-image-compressor)
   Batch compress images

## Office
1. [PrintOne](https://github.com/xtieume/PrinterOne)
   Share printer via TCP/IP in the local network with a dedicated IP and Port

### Windows annoyances
1. [Wedge](https://github.com/MarcGuiselin/wedge)
   Totally remove the edge browser from the default browser
2. [CopyLink to Clipboard](https://github.com/Avtera/CopyLink-to-Clipboard)
   Trick windows to copy the link instead of opening it

### Spotify things
1. [Spicetify](https://github.com/spicetify/cli)
   Spotify is a browser that enables it to add extensions
2. [Rollback Spotify](https://github.com/amd64fox/Rollback-Spotify)
   Choose specific spotify version to install then block the updates
3. [Block the Spot](https://github.com/mrpond/BlockTheSpot)
   Remove ads
